Добавить библиотеку netty-all-4.0.7.Final.jar;
Запустить HttpServer;
Зайти в браузер и ввести в командную строку адрес, например http://localhost:8080/hello
Программа работает по порту 8080

На обычные запросы типа http://localhost:8080/rjghejkrghejk и подобные вернет HTTP_1_1 OK.

Ключевые запросы:
http://localhost:8080/hello
http://localhost:8080/redirect?url=www.google.com
http://localhost:8080/status
